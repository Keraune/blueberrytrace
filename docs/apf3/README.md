# APF3 — BlueberryTrace

Esta carpeta reúne la documentación preparada para el **Avance de Proyecto Final 3** del curso Integrador I: Sistemas Software.

## Objetivo académico

El APF3 corresponde a la Unidad 3: Desarrollo. En esta etapa se debe demostrar que el proyecto ya cuenta con una arquitectura definida, avance funcional, organización del código, uso de recursos Java y control de versiones.

## Archivos incluidos

| Archivo | Uso recomendado |
| --- | --- |
| `01-informe-apf3.md` | Base del informe para Word/PDF. |
| `02-arquitectura-y-solid.md` | Sustento técnico de arquitectura, MVC/API-first, DAO/JPA, SOLID, TDD y seguridad. |
| `03-recursos-java.md` | Librerías y recursos Java aplicados en BlueberryTrace. |
| `04-git-github-evidencias.md` | Comandos y evidencias para repositorio GitHub. |
| `05-guion-exposicion-apf3.md` | Texto rápido para exponer el avance. |
| `06-checklist-apf3.md` | Checklist final antes de presentar. |
| `diagramas/*.mmd` | Diagramas Mermaid para pegar en Markdown, Draw.io o convertir a imagen. |
| `capturas/` | Carpeta sugerida para guardar capturas del sistema funcionando. |
| `evidencias/` | Carpeta sugerida para logs de Git, builds y comandos ejecutados. |

## Comando de apoyo

Desde la raíz del proyecto:

```bash
npm run apf3:evidence
```

El comando genera archivos de evidencia en `docs/apf3/evidencias/` con estado de Git, historial, ramas, estructura del proyecto y verificación básica de archivos clave.
