# BlueberryTrace — Fase 28

## Resumen

Esta fase refuerza la iconografía de estados, roles y clasificaciones usando badges con iconos y color semántico.
También se corrige visualmente el stepper de registro de siembra para evitar el subrayado/solapamiento de la línea con el texto de cada paso.

## Cambios principales

- Badges con iconos para roles, estados y categorías de clasificación.
- Colores diferenciados por rol: administrador, supervisor, operario y control de calidad.
- Colores e iconos diferenciados por estado operativo.
- Estados de usuarios reutilizando badges corporativos.
- Stepper de siembra con línea de progreso mejor alineada y sin cruzar el texto.
