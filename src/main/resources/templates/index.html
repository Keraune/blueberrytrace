Un front corporativo claro para supervisar invernaderos, camas y procesos

Plataforma interna enfocada solo en el área de arándanos dentro de frutales. El sistema organiza invernaderos como A1, A2, B1 y B2, permite controlar camas o filas de bandejas y mejora la visibilidad de siembra, procesos, clasificación y despacho.

Módulos visibles dentro del dashboard

El diseño se separa por procesos y gestión admin para mejorar la lectura y navegación.

Procesos

Invernaderos, camas, siembra, procesos, clasificación y despacho en una misma línea de trabajo.

Gestión admin

Usuarios, accesos y configuración del sistema en una sección administrativa independiente.

Reportes

Base visual lista para consolidar información por lote, fecha, clasificación o despacho.

Ajustes de cuenta

La cuenta del usuario se presenta como otro dashboard, no como una vista improvisada.

Información clave del alcance

Diseño alineado al contexto real del proyecto.

Alcance real

Solo cubre arándanos dentro del área de frutales. No abarca toda la empresa ni otras líneas productivas.

Estructura operativa

Los lotes se representan como invernaderos. Cada invernadero contiene varias camas o filas de bandejas.

Objetivo del rediseño

Mejor contraste, jerarquía visual, bloques seccionados, tarjetas limpias y mejor experiencia de uso.
